# 💰 Reajuste de Salário

Este projeto consiste na implementação de um algoritmo que calcula o
reajuste salarial de um colaborador com base em faixas de salário.

## 🚀 Tecnologias

-   JavaScript / Node.js
-   Lógica de programação

## 🎯 Objetivo

Sistema web para validação de dados cadastrais e cálculo automático de aumentos salariais, 
com processamento dinâmico de rotas HTTP e respostas em HTML.

## 📊 Regras de negócio

-   Até R\$ 400 → 15%
-   Até R\$ 800 → 12%
-   Até R\$ 1200 → 10%
-   Até R\$ 2000 → 7%
-   Acima → 4%

## ▶️ Como executar

node index.js
